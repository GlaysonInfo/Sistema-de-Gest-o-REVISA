Instalação para iOS

URL de Acesso

Exemplo: http://192.168.1.100:5176/

Como Instalar
1

Abrir no Safari

Cole a URL acima na barra de endereço do Safari e toque Enter

2

Carregar a Página

Aguarde até a página carregar completamente (cerca de 3-5 segundos)

3

Tocar no Botão Compartilhar

Toque no ícone 📤 (caixa com seta) na barra inferior do Safari

4

Encontrar "Adicionar à Tela Inicial"

Role a lista até encontrar "Adicionar à Tela Inicial" e toque

5

Confirmar Adição

Toque em "Adicionar" no canto superior direito

6

Pronto! 🎉

O app aparecerá na tela inicial. Toque para abrir.

Informações Importantes
⚠️ Primeira Vez no App?
Clique em "Checar API" para validar a conexão antes de fazer login.
✅ Funciona Offline
Mesmo sem internet, você pode usar o app normalmente. Os dados serão sincronizados quando voltar online.
Solução de Problemas
❓ "API offline" (vermelho)

1. Verifique se seu celular está na mesma rede WiFi do servidor
2. Altere o IP da URL para o IP correto do servidor
3. Clique "Checar API" novamente

❓ "Login inválido"

1. Confirme que digitou: admin
2. Confirme a senha: Admin@123
3. Clique "Checar API" primeiro para validar conexão

❓ "App não aparece na tela inicial"

1. Certifique-se de estar usando Safari (não Chrome)
2. Recarregue a página (puxe para baixo)
3. Tente novamente: Compartilhar → Adicionar à Tela Inicial

❓ "Sem internet - como usar offline?"

1. Use o app normalmente - ele funciona sem internet
2. Os dados são salvos no celular
3. Quando voltar online, toca para sincronizar
